# Как создать EXE файл

## Простой способ (Рекомендуется)

1. Запустите `build_exe.bat`
2. Дождитесь завершения (может занять 1-2 минуты)
3. EXE файл будет в папке `dist\VideoDownloader.exe`

## Продвинутый способ

1. Запустите `build_exe_advanced.bat`
2. Создаст EXE со всеми зависимостями
3. Файл будет больше, но более надежный

## Ручная сборка

```bash
# Установить PyInstaller
pip install pyinstaller

# Создать EXE
pyinstaller --onefile --windowed --name "VideoDownloader" video_downloader.py
```

## Параметры PyInstaller

- `--onefile` - один EXE файл (не папка)
- `--windowed` - без консоли
- `--name` - имя EXE файла
- `--icon` - путь к иконке (опционально)

## После сборки

EXE файл находится в папке `dist/`:
- `VideoDownloader.exe` - готовый к использованию
- Можно копировать куда угодно
- Не требует установки Python
- Работает на любом Windows 10/11

## Размер файла

- Простая сборка: ~15-20 MB
- С зависимостями: ~30-40 MB
- Это нормально - включает Python и все библиотеки

## Важно: FFmpeg

EXE НЕ включает FFmpeg. Для максимального качества:

### Вариант 1: Установить FFmpeg системно
```bash
winget install Gyan.FFmpeg
```

### Вариант 2: Положить рядом с EXE
1. Скачайте ffmpeg.exe
2. Положите в ту же папку что и VideoDownloader.exe

### Вариант 3: Без FFmpeg
Приложение будет работать, но:
- Скачает готовые форматы (обычно 720p-1080p)
- Не сможет объединять видео+аудио для 4K

## Распространение

Можете делиться EXE файлом:
- Отправить друзьям
- Загрузить на GitHub
- Поделиться в соцсетях

Пользователям НЕ нужно:
- Устанавливать Python
- Устанавливать библиотеки
- Настраивать что-либо

Просто запустить EXE!

## Антивирус

Некоторые антивирусы могут ругаться на EXE:
- Это ложное срабатывание
- PyInstaller упаковывает Python код
- Добавьте в исключения антивируса

## Обновление

Чтобы обновить EXE:
1. Измените `video_downloader.py`
2. Запустите `build_exe.bat` снова
3. Новый EXE заменит старый
