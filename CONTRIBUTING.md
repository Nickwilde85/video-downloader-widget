# Contributing to Video Downloader Widget

Спасибо за интерес к проекту! Мы рады любому вкладу.

## 🚀 Как внести вклад

### 1. Сообщить о баге

Нашли баг? Создайте [Issue](../../issues/new) с описанием:
- Что произошло
- Что ожидалось
- Шаги для воспроизведения
- Скриншоты (если возможно)
- Версия Windows и Python

### 2. Предложить улучшение

Есть идея? Создайте [Issue](../../issues/new) с тегом `enhancement`:
- Опишите идею
- Зачем это нужно
- Как это должно работать

### 3. Добавить код

1. **Fork репозитория**
2. **Создайте ветку**
   ```bash
   git checkout -b feature/my-feature
   ```
3. **Внесите изменения**
4. **Протестируйте**
5. **Commit**
   ```bash
   git commit -m "Add: описание изменений"
   ```
6. **Push**
   ```bash
   git push origin feature/my-feature
   ```
7. **Создайте Pull Request**

## 📝 Стиль кода

- Используйте 4 пробела для отступов
- Комментарии на русском или английском
- Понятные имена переменных
- Документируйте сложные функции

## 🧪 Тестирование

Перед PR убедитесь что:
- [ ] Приложение запускается
- [ ] Все функции работают
- [ ] Нет ошибок в консоли
- [ ] UI выглядит корректно во всех темах
