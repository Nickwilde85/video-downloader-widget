# Как добавить Video Downloader на рабочий стол

## Способ 1: Автоматическое создание ярлыка (Рекомендуется)

1. Запустите файл `create_desktop_shortcut.bat`
2. Ярлык "Video Downloader" появится на рабочем столе
3. Готово! Теперь можно запускать виджет с рабочего стола

## Способ 2: Ручное создание ярлыка

1. Кликните правой кнопкой мыши на `run.bat`
2. Выберите "Создать ярлык"
3. Перетащите ярлык на рабочий стол
4. Переименуйте в "Video Downloader"

## Способ 3: Закрепить в панели задач

1. Запустите `run.bat`
2. Когда приложение откроется, кликните правой кнопкой на иконке в панели задач
3. Выберите "Закрепить на панели задач"

## Способ 4: Автозапуск при старте Windows

1. Нажмите `Win + R`
2. Введите: `shell:startup`
3. Скопируйте ярлык `Video Downloader` в открывшуюся папку
4. Теперь виджет будет запускаться автоматически при входе в Windows

## Дополнительно: Изменить иконку

1. Кликните правой кнопкой на ярлыке → Свойства
2. Нажмите "Сменить значок"
3. Выберите понравившуюся иконку из системных или укажите свою

## Примечание

Виджет запускается без консоли благодаря использованию `pythonw` в `run.bat`
