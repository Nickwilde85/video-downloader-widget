# Поддерживаемые сайты / Supported Sites

Video Downloader поддерживает скачивание видео с **1000+ сайтов** благодаря библиотеке yt-dlp.

## 🎬 Популярные платформы

### ✅ Полная поддержка (работает отлично)

- **YouTube** - все форматы, плейлисты, 4K/8K
- **TikTok** - видео без водяных знаков
- **Instagram** - посты, reels, stories, IGTV
- **Facebook** - публичные видео
- **Vimeo** - все видео
- **Dailymotion** - все форматы
- **Reddit** - v.redd.it видео
- **Twitch** - клипы и VOD
- **SoundCloud** - аудио треки

### ⚠️ Частичная поддержка (могут быть ограничения)

- **Twitter/X** - публичные видео (приватные не работают)
- **Pinterest** - видео пины
- **LinkedIn** - публичные видео
- **Tumblr** - видео посты

### ❌ Не поддерживается

- **Netflix, Disney+, HBO** - защищены DRM
- **Приватные видео** - требуют авторизации
- **Платные курсы** - защищены авторскими правами

## 🔍 Как проверить поддержку сайта

1. Попробуйте скачать - если работает, значит поддерживается
2. Проверьте список на GitHub: https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md
3. Если не работает - возможно нужна авторизация или сайт изменил API

## 💡 Советы

### Для лучших результатов:

1. **Используйте прямые ссылки на видео**
   - ✅ `youtube.com/watch?v=...`
   - ❌ `youtube.com/playlist?list=...` (плейлисты могут не работать)

2. **Проверьте доступность видео**
   - Откройте ссылку в браузере
   - Убедитесь что видео публичное
   - Проверьте что видео не удалено

3. **Для Twitter/X**
   - Используйте полную ссылку на твит
   - Убедитесь что твит содержит видео (не GIF, не картинку)
   - Приватные аккаунты не поддерживаются

4. **Для Instagram**
   - Публичные аккаунты работают лучше
   - Stories могут быть недоступны после 24 часов

5. **Для максимального качества**
   - Установите FFmpeg: `winget install Gyan.FFmpeg`
   - Выберите качество "max"
   - Для 4K/8K обязательно нужен FFmpeg

## 🆘 Если не работает

1. **Обновите yt-dlp**
   ```bash
   pip install --upgrade yt-dlp
   ```

2. **Проверьте ссылку**
   - Откройте в браузере
   - Убедитесь что видео доступно

3. **Попробуйте другое качество**
   - Вместо "max" выберите "1080p" или "720p"

4. **Проверьте интернет**
   - Некоторые сайты блокируют загрузки

## 🔄 Обновления

yt-dlp регулярно обновляется и добавляет поддержку новых сайтов. Чтобы получить последние обновления:

```bash
pip install --upgrade yt-dlp
```

Затем пересоберите EXE файл если используете его.

## ⚖️ Легальность

**Важно**: Скачивайте только контент, на который у вас есть права:
- Ваши собственные видео
- Видео с разрешением автора
- Контент под свободной лицензией

Не используйте для:
- Пиратства
- Нарушения авторских прав
- Коммерческого использования без разрешения
